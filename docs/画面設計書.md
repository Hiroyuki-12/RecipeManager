# レシピ管理アプリ 画面設計書

## 1. はじめに
本書は、レシピ管理アプリの各画面の UI レイアウト・項目・操作を定義するものである。

---

## 2. 画面一覧

| 画面 ID | 画面名 | URL | 説明 |
|---------|--------|------|------|
| S-01 | レシピ一覧画面 | `/` または `/recipes` | レシピ一覧、検索、絞り込み、並べ替え |
| S-02 | レシピ詳細画面 | `/recipes/:id` | レシピの詳細表示 |
| S-03 | レシピ登録画面 | `/recipes/new` | 新規レシピの登録 |
| S-04 | レシピ編集画面 | `/recipes/:id/edit` | 既存レシピの編集 |

---

## 3. 共通レイアウト

```
┌──────────────────────────────────────────┐
│  🍳 レシピ管理アプリ                     │  ← ヘッダー(全画面共通)
├──────────────────────────────────────────┤
│                                          │
│           (ページごとのコンテンツ)       │
│                                          │
└──────────────────────────────────────────┘
```

- ヘッダーのタイトルをクリックすると一覧画面へ戻る

---

## 4. 画面詳細

### S-01 レシピ一覧画面

#### レイアウト
```
┌──────────────────────────────────────────────────┐
│  🍳 レシピ管理アプリ                             │
├──────────────────────────────────────────────────┤
│                                                  │
│  [🔍 レシピ名で検索 ____________]                │
│                                                  │
│  カテゴリ:[全て ▼]  並び順:[新しい順 ▼]         │
│                                  [+ 新規登録]    │
│  ────────────────────────────────────────        │
│  ┌────────────────────────────────────────┐      │
│  │ 肉じゃが                                │      │
│  │ 和食 / 30分                             │      │
│  └────────────────────────────────────────┘      │
│  ┌────────────────────────────────────────┐      │
│  │ カレー                                  │      │
│  │ 洋食 / 45分                             │      │
│  └────────────────────────────────────────┘      │
│  ┌────────────────────────────────────────┐      │
│  │ 麻婆豆腐                                │      │
│  │ 中華 / 20分                             │      │
│  └────────────────────────────────────────┘      │
│                                                  │
└──────────────────────────────────────────────────┘
```

#### 画面項目
| No | 項目 | 種別 | 動作 |
|----|------|------|------|
| 1 | 検索ボックス | テキスト入力 | 入力で部分一致検索を実行 |
| 2 | カテゴリプルダウン | プルダウン | 「全て / 和食 / 洋食 / 中華 / その他」 |
| 3 | 並び順プルダウン | プルダウン | 「新しい順 / 名前順」 |
| 4 | 新規登録ボタン | ボタン | 登録画面へ遷移 |
| 5 | レシピカード | カード | クリックで詳細画面へ遷移 |

#### 表示時の状態
- 初期表示:登録順(新しい順)で全件表示
- データなし時:「レシピがありません」と表示
- 検索結果なし時:「該当するレシピがありません」と表示

│  🍳 レシピ管理アプリ                             │
├──────────────────────────────────────────────────┤
│                                                  │

│   ・じゃがいも ・・・ 3個                        │
│   ・牛肉 ・・・・・・ 200g                       │
│                                                  │
│  【手順】                                        │
│   1. 材料を一口大に切る                          │
│   2. 鍋で炒める                                  │
│   3. 調味料を加えて煮込む                        │
│                                                  │

#### 画面項目
| No | 項目 | 種別 | 動作 |
|----|------|------|------|

#### ダイアログ
- 削除確認:「このレシピを削除しますか?」 [OK] [キャンセル]

### S-03 レシピ登録画面

#### レイアウト
```
┌──────────────────────────────────────────────────┐
│  🍳 レシピ管理アプリ                             │
├──────────────────────────────────────────────────┤
│                                                  │
│  新規レシピ登録                                  │
│  ────────────────────────────────────────        │
│                                                  │
│  レシピ名 *                                      │
│  [_______________________________]               │
│                                                  │
│  カテゴリ          調理時間    何人分            │
│  [和食 ▼]          [ 30 ]分    [ 4 ]人分         │
│                                                  │
│  材料 *                                          │
│  [玉ねぎ_______]  [1個_______]  [削除]           │
│  [豚肉_________]  [200g______]  [削除]           │
│  [+ 材料を追加]                                  │
│                                                  │
│  手順 *                                          │
│  1. [玉ねぎを薄切りにする_____________] [削除]   │
│  2. [フライパンで炒める_______________] [削除]   │
│  [+ 手順を追加]                                  │
│                                                  │
│  メモ                                            │
│  ┌────────────────────────────────────┐          │
│  │                                    │          │
│  │                                    │          │
│  └────────────────────────────────────┘          │
│                                                  │
│              [キャンセル]  [登録]                │
│                                                  │
└──────────────────────────────────────────────────┘
```

#### 画面項目
| No | 項目 | 種別 | 必須 | 説明 |
|----|------|------|------|------|
| 1 | レシピ名 | テキスト入力 | ○ | 最大 100 文字 |
| 2 | カテゴリ | プルダウン | - | 和食 / 洋食 / 中華 / その他 |
| 3 | 調理時間 | 数値入力 | - | 分単位、0 以上 |
| 4 | 何人分 | 数値入力 | - | 1 以上 |
| 5 | 材料行 | テキスト入力×2 | ○ | 材料名 + 分量、最低 1 行 |
| 6 | 材料追加ボタン | ボタン | - | 材料行を追加 |
| 7 | 材料削除ボタン | ボタン | - | 該当行を削除 |
| 8 | 手順行 | テキスト入力 | ○ | 最低 1 行、番号は自動採番 |
| 9 | 手順追加ボタン | ボタン | - | 手順行を追加 |
| 10 | 手順削除ボタン | ボタン | - | 該当行を削除 |
| 11 | メモ | テキストエリア | - | 複数行入力可 |
| 12 | 登録ボタン | ボタン | - | 入力チェック→登録 |
| 13 | キャンセルボタン | ボタン | - | 一覧画面へ戻る |

#### バリデーションエラー表示
- 各入力項目の下に赤字でエラーメッセージを表示
  - 「レシピ名を入力してください」
  - 「材料を 1 件以上入力してください」
  - 「手順を 1 件以上入力してください」

### S-04 レシピ編集画面

#### レイアウト
S-03 とほぼ同じ。タイトルが「レシピ編集」になり、ボタンが「更新」となる。
初期値として既存レシピの内容が表示される。

```
┌──────────────────────────────────────────────────┐
│  🍳 レシピ管理アプリ                             │
├──────────────────────────────────────────────────┤
│                                                  │
│  レシピ編集                                      │
│  ────────────────────────────────────────        │
│                                                  │
│  (S-03 と同じ入力フォーム、初期値あり)          │
│                                                  │
│              [キャンセル]  [更新]                │
│                                                  │
└──────────────────────────────────────────────────┘
```

#### 動作の差分
- 画面表示時に既存レシピを取得してフォームへセットする
- 更新成功時は詳細画面へ遷移

## 6. デザイン方針
- シンプル・ミニマルな UI
- レスポンシブ対応は本リリースでは対象外(PC ブラウザ前提)
- 配色は白基調、アクセントカラーは未定(実装フェーズで決定)
