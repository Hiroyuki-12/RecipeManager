# レシピ管理アプリ 画面遷移図

## 画面遷移図(Mermaid)

```mermaid
flowchart TD
    LIST["レシピ一覧画面<br/>(S-01)<br/>/recipes"]
    DETAIL["レシピ詳細画面<br/>(S-02)<br/>/recipes/:id"]
    NEW["レシピ登録画面<br/>(S-03)<br/>/recipes/new"]
    EDIT["レシピ編集画面<br/>(S-04)<br/>/recipes/:id/edit"]

    LIST -->|ヘッダー: + 新規登録| NEW
    LIST -->|レシピカードをクリック| DETAIL

    NEW -->|登録成功| DETAIL
    NEW -->|キャンセル| LIST

    DETAIL -->|編集ボタン| EDIT
    DETAIL -->|削除する → 確認| LIST
    DETAIL -->|← 一覧へ戻る| LIST

    EDIT -->|更新成功| DETAIL
    EDIT -->|キャンセル / ← 詳細へ戻る| DETAIL
```

## 画面一覧

| 画面 ID | 画面名 | URL |
|---------|--------|------|
| S-01 | レシピ一覧画面 | `/` または `/recipes`（`/` は `/recipes` へリダイレクト） |
| S-02 | レシピ詳細画面 | `/recipes/:id` |
| S-03 | レシピ登録画面 | `/recipes/new` |
| S-04 | レシピ編集画面 | `/recipes/:id/edit` |
